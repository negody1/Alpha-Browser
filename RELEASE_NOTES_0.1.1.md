# Alpha Browser v0.1.1

> Подготовлено. GitHub Release НЕ создан, тег НЕ запушен — только материалы для релиза.

## Что нового для пользователей

**Alpha Proxy стал стабильнее и понятнее**
- Прокси больше не «отваливается» после перезапуска браузера.
- Честный статус подключения: «Готов к работе» показывается только когда сайты
  реально открываются через Alpha Proxy. Ошибки описаны человеческим языком.
- Новый раздел **Настройки → Мой доступ**: статус, дата активации, версия
  профиля и браузера, проверка доступа и обновлений, ссылка на страницу доступа.

**Проще начать**
- Пошаговый мастер первого запуска.
- Обновлённые страница заявки на доступ и админ-панель в фирменном стиле Alpha.

**Документы и приватность**
- PDF открываются прямо во вкладке (масштаб, поиск, печать, скачивание).
- Открытие локальных файлов через `Ctrl+O`.
- Ссылки автоматически очищаются от рекламных меток (`utm_*`, `fbclid`, `gclid`…).
- Защита от утечки реального IP через WebRTC на PROXY-вкладках.

## Обновление
1. Скачайте `AlphaBrowserSetup.exe` из этого релиза (GitHub Releases).
2. Запустите установщик поверх текущей версии.
3. Настройки, профиль Alpha Proxy, история и закладки **сохранятся** —
   переустановка не требуется.

## Важно
- Установщик не подписан — Windows может показать SmartScreen («Подробнее» →
  «Выполнить в любом случае»).
- Если Alpha Proxy не подключается после обновления — откройте
  «Настройки → Мой доступ» → «Проверить доступ».
