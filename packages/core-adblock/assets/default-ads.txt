# Alpha Adblock — default ads/analytics list (MVP, lightweight)
#
# Format (separator may be ':' or '='):
#   domain:example.com    → block example.com and all subdomains
#   host:ads.example.com  → block this host (and its subdomains) only
#   contains:substring    → block if the URL contains this literal substring
#
# Scope (MVP): block well-known ad networks and analytics endpoints only.
# Deliberately conservative — main frames are never blocked (see engine),
# and we use host: (not domain:) for analytics subdomains of otherwise-legit
# sites so we never take down the parent site. No cosmetic / ABP filtering.
