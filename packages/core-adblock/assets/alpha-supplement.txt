! Title: Alpha Browser supplement
! Description: Small, conservative supplement of third-party ad/tracking hosts
!   that slipped past the main lists in real testing. ABP syntax, parsed into the
!   Ghostery engine alongside EasyList/etc.
! Safety rules:
!   - Every rule is scoped $third-party so a site's OWN first-party requests are
!     never affected (e.g. ads.vk.com is only blocked when embedded on other sites).
!   - The browser additionally NEVER cancels a mainFrame document (hard guard in
!     AdblockService), so none of these can block a page navigation.
!   - Domains that are load-bearing for site/app functionality are intentionally
!     NOT blocked and are listed at the bottom with the reason.

! INTENTIONALLY NOT BLOCKED — would break functionality (kept as docs):
!   youtubei.googleapis.com      — YouTube InnerTube API; blocking breaks YouTube
!   redirector.googlevideo.com   — video routing; blocking breaks playback
!   s.youtube.com                — YouTube logging tied to playback session
!   graph.facebook.com           — Facebook Graph API; breaks FB features/login
!   connect.facebook.net         — Facebook JS SDK; used for "Login with Facebook"
!   graph.instagram.com          — Instagram API
!   i.instagram.com              — Instagram CDN/API; breaks Instagram
!   ssl.p.jwpcdn.com             — JWPlayer library CDN; blocking breaks video players
! These are tracking-adjacent but load-bearing, so they are left ALLOWED.
