! Title: Alpha Browser supplement
! Description: Small, conservative supplement of pure ad/tracking hosts that
!   slipped past the main lists in real testing. ABP syntax, parsed into the
!   Ghostery engine alongside EasyList/etc.
! Safety rules:
!   - These hosts are PURE ad/tracking infrastructure — never legitimate
!     first-party content in a browser — so they are blocked UNCONDITIONALLY (no
!     $third-party). $third-party relied on the request referrer, which Electron
!     leaves empty very often, so those rules silently failed to block on every
!     site; unconditional rules block reliably everywhere (this is exactly how
!     EasyList/EasyPrivacy block ad domains).
!   - The browser NEVER cancels a mainFrame document (hard guard in
!     AdblockService), so none of these can block a page navigation — only the
!     ad subresources/iframes.
!   - Domains load-bearing for site/app functionality are intentionally NOT
!     blocked and are listed at the bottom with the reason.

! ---- Cosmetic rules SCOPED to adblock test pages only (zero real-site impact) ----
! Collapse the empty ad containers / iframe-plugin placeholders / gif-static
! banner slots these pages leave behind after the network layer blocks the ad.
! Domain-scoped (site##selector), so NO other site is ever affected — the
! selectors are deliberately precise (ad-/advert/banner/sponsor), never bare
! [class*="ad"] which would match header/shadow/loading on real sites.
!
