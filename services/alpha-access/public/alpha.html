Доступ к Alpha Proxy

Введите email, чтобы отправить заявку на доступ. После одобрения администратор выдаст код активации для Alpha Browser.

Что делать после отправки заявки?

  1. Дождитесь подтверждения от администратора.
  2. После одобрения вам передадут код активации.
  3. Откройте Alpha Browser.
  4. Перейдите в Настройки → Alpha Proxy.
  5. Введите email и код.
  6. Дождитесь статуса «Alpha Proxy готов к работе».

Если код не пришёл: код выдаётся администратором после одобрения заявки. Если заявка уже одобрена, обратитесь к администратору.

Скачать Alpha Browser

Alpha Browser распространяется через GitHub Releases. Это снижает нагрузку на сервер Alpha Access: установщик скачивается напрямую с GitHub, а 3d.negody.ru отвечает только за регистрацию и выдачу доступа.

Если релиз ещё не опубликован, установщик будет доступен после публикации первой версии.

Как работают обновления

Alpha Browser сам проверяет GitHub Releases. Когда появляется новая версия, в браузере открывается уведомление с описанием изменений и кнопкой для перехода на страницу релиза. Обновление устанавливается вручную: пользователь скачивает новый установщик с GitHub и запускает его.

Как получить доступ

  1. Скачайте и установите Alpha Browser.
  2. Введите email на этой странице.
  3. Дождитесь подтверждения заявки.
  4. Получите код активации от администратора.
  5. Откройте Alpha Browser.
  6. Перейдите в настройки Alpha Proxy.
  7. Введите email и код активации.
  8. После активации Alpha Proxy подключится автоматически.
